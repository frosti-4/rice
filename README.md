# driftwm rice

Легковесный райс driftwm. В первую очередь оптимизирован для работы на слабом железе, поэтому упор сделан на минимализм и скорость, а не на тяжелые анимации и визуальные эффекты.

---

## Зависимости

Для корректной работы всех компонентов интерфейса вам понадобятся следующие пакеты:

* `driftwm` — оконный менеджер
* `waybar` — статус-бар
* `alacritty` — терминал
* `rofi` — меню запуска приложений
* `awww` — обои / фон
* `swaync` — центр уведомлений
* `swaylock-effects` — экран блокировки

> **Примечание:** Список зависимостей может быть неполным. Если чего-то не хватает, сверяйтесь со скриптом установки.

---

## Установка

Для установки конфигурации выполните следующие команды в терминале:

```bash
git clone [https://github.com/frosti-4/dots.git](https://github.com/frosti-4/dots.git)
cd dots
chmod +x install.sh
./install.sh
```
# Важное замечание

Скрипт создания конфигурации использует симлинки. Чтобы настройки не сломались, не удаляйте и не перемещайте склонированную папку dots после завершения установки.
